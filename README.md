## Features

- New string functions: title, capitalize, tochararr, _isupper, _islower, isnumeric

## Functions

- `title`: Returns a string with the first character of each word capitalized.
- `capitalize`: Returns a string with the first character capitalized and the rest lowercased.
- `tochararr`: Converts a string to an array of characters.
- `_isupper`: Checks if all characters in the string are upper case.
- `_islower`: Checks if all characters in the string are lower case.
- `isnumeric`: Checks if all characters in the string are numeric.
